\section*{Purpose}
Modular documentation for the DSPRO2 Pok\'{e}mon reinforcement-learning project. Sources:
\begin{itemize}
  \item \texttt{temp.md} --- project narrative and a week-7 snapshot paragraph set;
  \item \texttt{temp/ai\_docs/*.md} --- architecture, pipeline, model, challenges, phased experiments (local only; Git ignores \texttt{temp/});
  \item \texttt{CLAUDE.md}, \texttt{README.md}, and Python sources under \texttt{src/}.
\end{itemize}

\section*{Relation to \texttt{RL\_Data\_Sheet.tex}}
\texttt{Doc/RL\_Data\_Sheet.tex} is the DSPRO2 ``RL project sheet'' with structured sections (goal, environment, reward, agent, training, results). This report is a longer-form draft: milestones, challenge model, implementation evolution, and internal engineering notes. Where the two disagree, prefer \textbf{this} document's implementation chapters (tied to current code) and treat the factsheet as a parallel hand-in you may refresh separately.
